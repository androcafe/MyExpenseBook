# MyExpenseBook

This android application consist of :
1. store our daily expenses
2. retrieve our monthly expense
3. check our saving according to monthly income

In the terms of android methodologis project consist use of :
1) SQLite Database
2) SharedPreference
3) Custom Dialog box
4) Custom Spinner
5) Custom Edittext
6) Custom Button
7) Menus
